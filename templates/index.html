JSON Format Help

            

                
Rectangular Board Format:

                
{
  "rows": 4,
  "cols": 7,
  "regions": [...]
}

                
                
Constraint Types:

                
    
                    
  • "type": "=" - All dots equal
    • 
                    
  • "type": "!=" - All dots different
    • 
                    
  • "type": ">", "value": N - All dots greater than N
    • 
                    
  • "type": "<", "value": N - All dots less than N
    • 
                    
  • "type": "sum", "value": N - Dots sum to N
    • 
                    
  • "type": "none" - No constraint
    • 
                

                
                
Optional Dominoes:

                
{
  "rows": 2,
  "cols": 4,
  "dominoes": [[0, 0], [0, 1], [1, 2], [3, 4]],
  "regions": [...]
}
